Java desktop emulator

GameDuck

GameDuck is a desktop Game Boy and Game Boy Color emulation suite written in Java with a modern Swing frontend. The v0.3.0 major update introduces cheat support, a complete UI redesign, native dark mode, and significant stability enhancements.

Built in 2024-2026
Status Public alpha
Role Solo developer
Download v0.3.0-alpha View source code
Latest public release: v0.3.0-alpha Targets Java 22 and Maven Supports Game Boy and Game Boy Color

Release focus

Modern and Feature-Rich

Version 0.3.0 marks a shift towards a more modern user experience, bringing proper dark mode, integrated cheat support, and refined library tools to the desktop platform.

Project Snapshot

Stack

Java 22 + Swing

GameDuck remains portable and lightweight, built with Java 22 and a custom-themed Swing UI that now supports advanced rendering effects.

Scope

GB / GBC / ROM Hacks

Full support for original hardware software, with dedicated workflows for ROM hacks and real-time IPS patching.

Current release

v0.3.0-alpha

The latest build includes Cheat Support, Dark Mode, Frame Blending, and a redesigned Audio Enhancement suite using compact knobs.

Build Breakdown

Why I built it

Pushing the limits of Swing and Java

Initially an academic project, GameDuck has evolved into a showcase for building highly performant, modern desktop applications using "legacy" Java technologies.

Cheat Engine

Automated cheat downloads and management

The new cheat system allows users to fetch cheats for specific ROMs automatically, enabling real-time toggling and editing within the UI.

Accuracy & Performance

Stable emulation with modern visuals

Version 0.3.0 fixes long-standing issues with window maximizing, input lag, and audio stuttering, while introducing frame blending for smoother motion.

Lessons learned

User Experience is as vital as accuracy

Developing GameDuck proved that an emulator is only as good as its interface; modern expectations like Dark Mode and easy shortcuts are essential.

What's next

Hardware expansion and link cable

The roadmap focuses on Super Game Boy support, internal link cable emulation, and identifying remaining sources of UI choppiness during high-load.

What GameDuck Includes

Emulation

Cheat Support & Frame Blending

Beyond basic GB/GBC play, GameDuck now supports cheats, FPS counters, and frame blending to mimic classic screen ghosting.

UI / Design

Modern UI with Native Dark Mode

A total visual overhaul brings a cleaner layout, better scaling for large icons, and a dedicated dark theme for users.

Library

Persistent filtering and managed ROMs

The library now remembers your viewing preferences and supports one-click ROM hack loading with automatic metadata caching.

Input

Controller-mapped shortcuts

Application shortcuts (save, load, fast-forward) can now be mapped directly to controller buttons for a seamless couch-play experience.

Themes

JSON Theme Storage & Quick Editing

Custom themes are now portable via JSON, and the UI includes clickable swatches for instant visual feedback during customization.

Project context

A Professional Desktop Suite

GameDuck has moved far beyond its dissertation roots. It is now a robust, packaged application that prioritizes user workflow— from library management and cheat integration to controller-first shortcuts and visual customization.

Modern Desktop UX

The v0.3.0 update proves that Swing can deliver a premium look and feel when paired with thoughtful design.

Portable Release Model

Available as a standalone x64 Windows build with a bundled runtime—no Java install required.

Identity

The duck theme remains a tribute to the campus life at Lancaster University where the core logic was born.

Roadmap Toward v1.0

Hardware

SGB and GBA support

Implementation of SGB borders and hardware features to expand the library of compatible experiences, as well as GBA games to complete the suite.

Optimization

Full Stability

Removing remaining choppiness when running the full suite of shaders, borders, and audio effects simultaneously.

Connectivity

Dual Emulation

Enabling dual emulation instances with internal linking for multiplayer support and trading.

(The roadmap has been updated to reflect features recently moved to stable in v0.3.0.)

Important Notes

GameDuck does NOT ship with commercial ROMs or Nintendo boot ROM files.

Boot ROM support is available, but users need to supply their own legal dumps separately.

The project is a personal emulator suite and is not affiliated with Nintendo, nor does it contain any copyright material.
